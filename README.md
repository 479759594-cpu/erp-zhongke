# 中科数控售后维修管理系统

广东中科数控科技有限公司 · 设备报修及配件购买售后管理平台

## 系统架构

- **客户服务中心** (`index.html`)：设备报修、配件选购、订单中心
- **售后管理后台** (`admin.html`)：工单管理、配件管理、出库订单、客户管理、账号管理、操作日志、系统设置
- **数据服务** (`server.js`)：Node.js HTTP 服务器，端口 8080，JSON 文件存储

## 快速启动

```bash
# 安装 Node.js 后直接运行
node server.js

# 或双击 start.bat（自动启动服务器 + 公网隧道）
```

## 功能特性

- **设备报修**：12种设备类型，级联故障部位选择，GPS定位，现场拍照上传
- **配件选购**：购物车模式，在线下单，二维码收款联动
- **订单跟踪**：下单→付款→发货→签收全流程状态时间线
- **工单管理**：接单/指派/报价/审批/维修时间/费用核算/保修单打印
- **配件管理**：库存管理、入库/出库、图片上传
- **客户管理**：级联省市区选择（34省全覆盖）、账号状态管理
- **账号管理**：多角色权限（管理员/售后人员）、编辑信息、密码重置
- **操作日志**：完整审计记录、类型筛选、日期范围查询
- **保修判断**：自动计算保修期（常规12月/激光器24月）
- **安全机制**：PBKDF2-SHA512密码哈希、Bearer Token认证、路径遍历防护

## 技术栈

- 前端：原生 HTML/CSS/JS，无框架，无 CDN 依赖
- 后端：Node.js 原生 HTTP 模块
- 数据：JSON 文件存储，自动备份恢复
- 安全：PBKDF2 密码哈希，Token 认证，CORS 控制

## 数据文件

- `data.json`：主数据文件（管理员、客户、工单、配件、订单、设置）
- `audit.json`：审计日志
- `.bak` 文件：自动备份
